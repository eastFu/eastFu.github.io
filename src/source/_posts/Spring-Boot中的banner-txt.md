## 关于banneer
banner是spring boot 应用启动时打印在控制台的彩蛋信息，默认是这样的
```
.   ____          _            __ _ _
 /\\ / ___'_ __ _ _(_)_ __  __ _ \ \ \ \
( ( )\___ | '_ | '_| | '_ \/ _` | \ \ \ \
 \\/  ___)| |_)| | | | | || (_| |  ) ) ) )
  '  |____| .__|_| |_|_| |_\__, | / / / /
 =========|_|==============|___/=/_/_/_/
 :: Spring Boot ::        (v1.4.3.RELEASE)
```

## DIY
想要修改这个文本的话，只需要在resources下新建banner.txt即可。这里可以自定义banner： http://patorjk.com/software/taag
